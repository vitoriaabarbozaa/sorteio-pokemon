Este é um aplicativo web simples desenvolvido com foco na visualização mobile (Mobile First), que consome e exibe dados da [PokéAPI](https://pokeapi.co/). O objetivo principal deste projeto é praticar o consumo de APIs públicas, requisições assíncronas e manipulação do DOM.

## 📱 Visão Geral
O usuário clica em um botão e o sistema gera um ID aleatório (de 1 a 1025). A partir desse número, uma requisição `fetch` busca os dados correspondentes na PokéAPI e exibe o card do Pokémon na tela com suas principais características.

## 🚀 Funcionalidades
* Sorteio aleatório de um Pokémon da base de dados.
* Exibição da imagem, nome e número (ID).
* Listagem dinâmica dos tipos (ex: fogo, água, planta).
* Exibição dos status base do Pokémon: HP, Attack, Defense, Special-Attack, Special-Defense e Speed.
* Reprodução do áudio (grito) oficial do Pokémon sorteado.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica do projeto.
* **CSS3:** Eu adicionei o CSS base para o funcionamento e entrega, mas pretendo aperfeiçoar mais futuramente. Por enquanto, ta concluído, mas talvez eu volte a mexer nele.
* **JavaScript:** Lógica principal, geração de números aleatórios e consumo da API nativo do navegador.
